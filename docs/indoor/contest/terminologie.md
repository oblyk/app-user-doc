# Términologie

Voici les termes que nous utilisons pour parler des différentes parties d'un contest

**Catégorie :**  
Les catégories représentent une classe d'aĝe comme 10/14 ans, ou une typologie de participant : Loisir / Expert.
Le classement fera la distinction entre les femmes et les hommes automatiquement, vous n'avez pas besoin de faire une catégorie par âge et par genre.

**Les vagues :**  
Vous pouvez définir un nombre de vagues qui vous permettra de donner des créneaux de passage différent sur les qualifications par exemple.

**Les épreuves :**  
Les épreuves sont les grandes sections d'un combiné, par exemple sur un combiné classique, il y aura une épreuve de voie, une épreuve de bloc et une épreuve de vitesse.
Chaque épreuve peut être divisée en étape, exemple Bloc - Qualification, Bloc - Finale, Voie - Qualification, Voie - Finale, etc.

Un contest, même si ce n'est pas un combiné a forcément une épreuve.

**Les étapes :**  
Classiquement les étapes sont les : Qualification / Demi-finale / Finale.
Vos pouvez définir autant d'étape que vous voulez, les participants sont automatiquement "inscrit" à la première étape de chaque épreuve. Par contre, vous devrez les inscrire vous-même aux étapes suivantes suivant leurs performances dans les étapes précédentes.

**Les groupes de lignes :**  
Dans chaque étape vous pouvez ajouter des blocs/voie qui devront être fait par la catégorie et/ou genre que vous souhaitez.  
Par exemple, vos 20 blocs de qualifications vont êtres partagés par toutes vos catégories et genre, par contre sur les finales, vous aurez sûrement : 3 blocs jeune femme, 3 blocs jeune homme, 3 blocs séniors femme et 3 blocs séniors homme.

**Les championnats :**  
Le championnat vous permet de lier vos différents contest entre eux pour avoir un classement sur l'année par exemple.
